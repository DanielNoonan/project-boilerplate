1). Make sure Node.js is installed globally on the computer.

2). Create a Git Repository and copy the CONTENTS of the 'project-starter' folder into it.

3). Open 'package.json' file and change the 'name' field appropriately.

4). Install all the dependencies by navigating to the new project folder in the command line and typing: npm install

5). Add flex-grid, flex-nav if required.

6). Carry out the initial 'Commit' and 'Git Push' to the 'Master Branch'.
